Installation

Installation is straightforward and follows open source conventions. The whole installation process should take no more than 5 minutes.

1. Requirements

Flod requires the following toolchain:

  • git 1.7 or later
  • cmake 2.8.0 or later
  • make
  • g++ or equivalent compiler
  • Perl 5.8 or later
  • sh

If a project has requirements beyond those listed above, then building and testing that project will only be possible if those requirements are met.

2. Download

Currently you need to build Flod from source. This means getting a copy of the source repository using git. Binary packages and source tarballs will be available later. 

$ git clone https://git.tasktools.org/scm/ts/flod.git flod.git
...

3. Build

Build Flod just as you would build any cmake-based project. 

$ cd flod.git
$ cmake .
...
$ make
...

If you see any errors or warnings, please report them to support@flod.us.

4. Test

$ make test
...

If you see any errors or warnings, please report them to support@flod.us.

5. Install

Typical installation requires root. 

$ sudo make install
...

Test your installation, by checking the version number. You should see no errors. 

$ flod --version
