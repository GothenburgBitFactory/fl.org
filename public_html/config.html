Configuration

Configuration settings are manipulated by the 'config' command. There are four variations: 

$ flod config name value
$ flod config name ''
$ flod config name
$ flod config

The first command, with both a name and a value will create or overwrite the setting. The second command, with a blank value will delete any existing value, but retain the setting. The third will delete the entry from the configuration file. The fourth command, with no name or value, will enumerate all the stored name/value pairs.

central.work.algorithm first This is the algorithm by which central selects from the list of available work. Currently only the 'first' algorithm is available.
central.work.obsolete 1month

Code changes become 'obsolete' after this period, and the code change is considered locked. All this means is that satellites will not be asked to build and test any code changes older than this.

A missing value means no code changes are obsoleted.

central.work.timeout 1hour

If work is not completed withing the timeout period, then it is considered lost, and will become available for build/test again.

A missing value means no timeout.

satellite.allow task.git,flod.git
all

This is a comma-separated list of projects for which the satellite will perform builds and run tests. A project must either be specifically allowed, or allowed via the 'all' value.

Note that a 'deny' value overrides an 'allow' value.

satellite.delay
satellite.deny task.git,flod.git
all
none

This is a comma-separated list of projects for which the satellite will not perform builds and run tests. If a project is denied by name, it will not be built. The value 'all' will prevent all activity.

Note that a 'deny' value overrides an 'allow' value.

satellite.label Ubuntu_12.10_64

A description of the satellite platform. It is recommended that the distribution name, version and 32/64 information is included, as these are likely identify the different satellites.

Any underscores in the value will be converted to spaces to allow text wrapping.
